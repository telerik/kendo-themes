# Daterangepicker Accessibility

## ARIA Attributes

The following table lists the ARIA attributes applied to the Daterangepicker component:

| Selector | Attribute | Usage |
| --- | --- | --- |
| `.k-daterangepicker .k-input-inner` | `label for or aria-label or aria-labelledby` | Each date input needs an accessible name. |

## Keyboard Navigation

The following table lists the keyboard shortcuts supported by the Daterangepicker component:

| Shortcut | Behavior |
| --- | --- |
| `Esc` | Closes the popup |
| `Alt/Opt(Mac) + Down` | Opens the popup |
| `Alt/Opt(Mac) + Up` | Closes the popup |
