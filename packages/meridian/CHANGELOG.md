### BREAKING CHANGES

* **meridian:** Starting with this release, @progress/kendo-theme-meridian
is the new recommended theme for Kendo and Telerik products. It replaces
@progress/kendo-theme-default as the default starting point for new
applications.

@progress/kendo-theme-default is now in maintenance mode. It will continue
to receive bug fixes, styles for new components, and compatibility updates,
but will no longer receive visual redesigns or new design concepts.

Existing applications can continue using Kendo Default theme and migrate
to Meridian on their own schedule. To learn more about this change, please
refer to the dedicated breaking change article in the Progress Design
System documentation.
