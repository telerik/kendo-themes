# ToggleButton Keyboard Support

## Managing Focus

The button is focusable by default, and follows the regular tab order of the page.

## Keyboard navigation

| Shortcut| |
|---------------------|---------------------|
|`Enter \ Space`| Selects/Deselects the button|
