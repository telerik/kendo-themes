# Pager Keyboard Support

## Managing focus
- Initial focus - First actionable item in the pager
- Moving tab index components strategy implemented.

## Keyboard Shortcuts

| Shortcut | Behavior |
|----------|----------|
| `Arrow Keys` | Moves focus to page buttons|
| `Enter`| Selects the page|
