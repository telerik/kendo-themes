# Captcha Keyboard Support

## Managing Focus

Tab Sequence follows the dom elements order. The focus order is as follows:

1. Captcha image
2. Reset button
3. Audio button
4. Input

When audio button is clicked (`enter` or `space`) the volume control (Slider component) is focused.
